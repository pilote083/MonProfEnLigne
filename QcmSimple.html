L'utilisation de l'auxiliaire 'do' au présent en Anglais

L'auxiliaire 'do' est utilisé en anglais pour former les questions, les négations et pour insister sur une action.

Négation

  • I => do => not => (I don't)
  • You => do => not => (You don't)
  • He/She/It => does => not => (He/She/It doesn't)
  • You => do => not => (You don't)
  • We => do => not => (We don't)
  • They => do => not => (They don't)

Question

  • Auxiliaire => sujet => verbe => (Auxiliaire + sujet + verbe)
  • Do => I => verbe => (Do I + verbe)
  • Do => you => verbe => (Do you + verbe)
  • Does => He/She/It => verbe => (Does he/she/it + verbe)
  • Do => you => verbe => (Do you + verbe)
  • Do => we => verbe => (Do we + verbe)
  • Do => they => verbe => (Do they + verbe)

Question négative

  • Auxiliaire => sujet => verbe => (Auxiliaire + sujet + verbe)
  • Don't => I => verbe => (Don't I + verbe)
  • Don't => you => verbe => (Don't you + verbe)
  • Doesn't => He/She/It => verbe => (Doesn't He/She/It + verbe)
  • Don't => you => verbe => (Don't you + verbe)
  • Don't => we => verbe => (Don't we + verbe)
  • Don't => they => verbe => (Don't they + verbe)

L'auxiliaire 'do' dans les questions

On utilise 'do' ou 'does' au début de la phrase pour poser une question à tous les temps simples du présent et du passé.
Auxiliaire + Sujet + Verbe
Exemples:
-Do you like pizza? (Aimes-tu la pizza?)
-Does she play tennis? (Joue-t-elle au tennis?)
-Does he work on Saturdays? (Travaille-t-il les samedis?)

L'auxiliaire 'do' dans les phrases négatives

On ajoute 'do not' (don't) ou 'does not' (doesn't) pour faire une négation.
Exemples:
-I do not like vegetables => I don't like vegetables. (Je n'aime pas les légumes.)
-He does not understand => He doesn't understand. (Il ne comprend pas.)
-She does not like red meat => She doesn't like red meat. (Elle n'aime pas la viande rouge)
